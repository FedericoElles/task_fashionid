
# Homework task


## Design goals

* Possibility to add further questions with single line of code for code and tests
* Reuse existing libraries
* Top-down readabilty of source code

## Installation

    git clone https://github.com/FedericoElles/task_fashionid.git
    cd task_fashionid
    npm install

## Test

    npm test
   

## Run
    
    npm start
